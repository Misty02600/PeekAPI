# Project Brief - PeekAPI

## 项目概述

PeekAPI 是一个本地 API 服务，提供屏幕截图和系统音频录制功能，通过系统托盘进行管理，支持公开/隐私模式切换。

## 核心需求

1. **屏幕截图服务** - 提供 HTTP API 获取当前屏幕截图，支持高斯模糊处理
2. **音频录制服务** - 持续录制系统音频（Loopback），提供最近录音的获取接口
3. **隐私保护** - 支持公开/私密模式切换，私密模式下拒绝所有请求
4. **系统托盘管理** - 通过系统托盘图标进行模式切换、重启录音等操作
5. **API 密钥保护** - 低模糊度截图需要 API 密钥验证

## 项目目标

- 为远程监控/机器人等应用提供本地屏幕和音频数据接口
- 简单易用，开箱即用
- 支持打包为独立 exe 文件
- 跨平台支持（计划中）

## 目标用户

- 需要远程获取电脑屏幕截图的开发者
- 聊天机器人开发者（如 ChieriBot）
- 需要本地音频监控的应用

## 成功标准

1. API 稳定可靠，响应迅速
2. 隐私模式切换即时生效
3. 音频录制线程健康运行，支持自动重连
4. 支持打包为独立可执行文件

## 项目边界

### 包含
- Windows 平台支持
- 屏幕截图（支持多显示器）
- 系统音频录制（Loopback）
- HTTP API 服务
- 系统托盘管理

### 不包含（当前）
- Linux/macOS 平台支持（计划中）
- 视频录制
- 远程控制功能
